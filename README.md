Special calculator for my fiend

Калькулятор считает сумму цифр в числе. Выводит последние три значения.

Игнорируется любой ввод кроме цифр от 1 до 9.

Если скопировать набор символов в input, то автоматически удалится все кроме цифр от 1 до 9.

Например:
1)input = 23801 => calculate = 2+3+8+1 = 14 => 1+4 = 5 => result = 14 5
2)input 1ff8abc@%3 => calculate = 1+8+3 ...

Сохранить лог можно двумя способами:
1) Нажать Enter
2) Кликнуть на надпись "Сохранить", которое появится при наведении.
